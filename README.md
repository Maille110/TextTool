# TextTool

Outil de manipulation de texte en ligne de commande.

## Fonctionnalités
- **uppercase** : Met le texte en majuscules.
- **lowercase** : Met le texte en minuscules.
- **length** : Compte le nombre de caractères.
- **prefix** : Garde les 10 premiers caractères.

## Utilisation
Lancer avec `python texttool.py`, puis taper une commande suivie d'un texte.
